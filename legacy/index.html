사용법

  • 좌표계: 툴바에서 직교 / 극좌표 전환 (단축키 1, 2)
  • 직교 — 함수 그리기: x 방향으로 한 번에 스케치. f(x)=x^2, A=(3,4)
  • 극좌표 — 극곡선: θ가 한 방향으로 증가하도록 스케치. r=1+cos(θ), A=(2; 60°)
  • 근사 방법: 툴바에서 다항식, sin/cos 조합, 지수, 거듭제곱, 로그, 일차, 자동(최적) 중 선택. 스케치 완료 시 적용되며, 변경 시 기존 스케치를 다시 근사합니다.
  • 이동: 드래그로 팬, 휠로 확대/축소 (커서 기준)
  • LaTeX 입력: 유형·이름 선택 후 LaTeX 수식 입력. 미리보기는 KaTeX로 렌더링. Enter 또는 [입력] 실행.
  • 단축키: M 이동, F 그리기, P 점, I 입력 포커스, Ctrl+Z/Y, Ctrl+↑/↓ 입력 기록
